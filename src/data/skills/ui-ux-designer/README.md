# UI/UX Designer & Creative Developer

Design-focused skill template for agents that bridge the gap between aesthetics and usability — designer-developers who code their own designs.

## What This Skill Does

Equips your agent with end-to-end design capabilities — from user research and Figma prototypes to production-ready React + Tailwind implementations with polished micro-interactions.

## Skills Covered

- End-to-end product design and user research
- High-fidelity prototyping in Figma
- Design systems — component libraries and design tokens
- Motion design with Framer Motion, Lottie, CSS animations
- React + Tailwind CSS frontend implementation
- Responsive, pixel-perfect UI development

## Ideal For

Designers who ship code — turning wireframes into production-ready interfaces with thoughtful micro-interactions and polished visual details.

## Installation

Copy this folder into your project's skills directory:

```
your-project/
└── skills/
    └── ui-ux-designer/
        ├── SKILL.md
        └── README.md
```
