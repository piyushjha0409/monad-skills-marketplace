# Full Stack Web3 Developer

Skill template for building production apps in the Web3 space, covering the entire stack from smart contracts to polished frontends.

## What This Skill Does

Equips your agent with full stack Web3 capabilities — React/Next.js frontends, Node.js backends, Solidity smart contracts, and seamless wallet integrations. Optimized for the Monad ecosystem.

## Skills Covered

- React, Next.js, Tailwind CSS, Zustand state management
- Solidity with Foundry and Hardhat tooling
- ERC-20/721/1155 token standards and gas optimization
- Node.js, Express, PostgreSQL, Redis backends
- Real-time WebSocket integrations
- Wallet connection flows with Viem/Wagmi
- Transaction handling and subgraph queries

## Ideal For

Builders who own the full stack — from writing and deploying smart contracts to shipping pixel-perfect frontends with seamless wallet integrations.

## Installation

Copy this folder into your project's skills directory:

```
your-project/
└── skills/
    └── fullstack-web3-dev/
        ├── SKILL.md
        └── README.md
```
