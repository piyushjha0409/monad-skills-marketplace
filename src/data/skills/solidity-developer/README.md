# Solidity Developer

Skill template for blockchain developers specializing in Solidity and smart contract development on EVM-compatible chains.

## What This Skill Does

Equips your agent with deep knowledge of Solidity smart contract development — from writing and testing contracts to gas optimization and security auditing. Optimized for the Monad ecosystem.

## Skills Covered

- Solidity 0.8+, ERC-20/721/1155 token standards
- Upgradeable proxies (UUPS, Transparent), diamond pattern
- Foundry (forge, cast, anvil) and Hardhat tooling
- Unit, fuzz, and invariant testing with coverage analysis
- Security patterns — reentrancy guards, access control, OpenZeppelin
- Gas optimization — storage packing, Yul assembly, batch ops
- DeFi primitives — AMMs, lending pools, vaults, oracles
- Multi-chain deployment and contract verification

## Ideal For

Builders focused on the smart contract layer — writing secure, gas-efficient Solidity for DeFi protocols, NFT platforms, DAOs, or any on-chain application on Monad and other EVM chains.

## Installation

Copy this folder into your project's skills directory:

```
your-project/
└── skills/
    └── solidity-developer/
        ├── SKILL.md
        └── README.md
```
